# Grill Me

Interview the user relentlessly about every aspect of this plan until we reach a shared understanding. Walk down each branch of the design tree, resolving dependencies between decisions one-by-one.

## Rules
- Ask 2-3 related questions at a time. Never more than 3.
- For each question, provide your recommended answer.
- If a question can be answered by exploring the codebase, explore the codebase instead of asking.
- Don't stop until every branch has been explored.
- Challenge weak answers. "Are you sure?" is valid.
- Track which branches are resolved vs open.
